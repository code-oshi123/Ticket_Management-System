# Event Booking & Management System 🎟️

A desktop-based Event Booking and Management System developed using **C#, .NET 8, Windows Forms, and MySQL**.  
This project was created as part of a university assignment to manage events, locations, and ticket bookings through a role-based system.

---

## 📌 Overview

The system allows **Admins, Organizers, and Users (Attendees)** to interact with the platform based on their roles.

- **Admin** – Manage events, locations, and feedback
- **Organizer** – Create and manage events
- **User** – Register, view events, and book tickets

The application integrates with a **MySQL database** to store all event, user, and ticket information.

---

## ✨ Features

- Role-based login system (Admin, Organizer, User)
- Event creation and management
- Ticket booking system
- Event location management
- User registration and password reset
- Feedback and review system
- Event scheduling with time-slot validation
